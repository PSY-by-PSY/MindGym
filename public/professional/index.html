個案旅程 · 一個月教練週期
這個 MVP 把四個 AI 模組串成一條完整的教練服務旅程。依序完成每一站，最後在「結案回顧」看到 AI 整合所有資料生成的前後比較與成長報告。所有資料儲存在你的瀏覽器中（localStorage），不會上傳到任何伺服器；AI 分析經由你自己的 Claude API Key 直接呼叫 Anthropic API。
知情同意書 個案尚未簽署
個案在第 1 站開始對話前，會先閱讀並簽署這份同意書。以下是預設範本（會自動帶入你的教練署名），你可以直接編輯成自己的版本——【】內的部分請替換成你的實際資訊。
快速開始
1. 點右上角「設定」，貼上你的 Anthropic API Key（到 Console 取得）並測試連線。
2. 每一站都有「填入示例」按鈕，可以不用自己打字、快速跑完整條旅程。
3. 想重新測試時，可在設定中清除全部資料。

結案前的資料盤點
結案回顧會整合整個週期的三種資料：初始評估（T0）、晤談分析、每日日記，再加上一次同量表的再測（T1），由 AI 生成前後比較與整體成長報告。
T1 再測 · 與 T0 相同的質性題目
個案再次回答一個月前的同一組問題。答完後 AI 會特別比較前後差異。
整體成長報告 · T0 → T1
AI 整合再測比較、晤談分析與日記全紀錄生成的結案報告。教練版聚焦專業判讀，個案版是一份值得收藏的「畢業快照」。
