## 0. 一句話目標

把市集從「服務者視角的四大類」改成「個案語言的狀態入口」，讓心理師的風格
可被看見、可被計算配對，並用「試玩小練習」讓個案在預約前就體驗到
「這位夥伴的東西對我有用」。

## 1. 已拍板決策

1. **頂層分類改為狀態/議題白話入口**：「睡不好」「情緒好滿」「壓力爆表」等
   個案語言；服務類型（心理諮詢/生涯教練/身心靈/測驗量表）降為次篩選。
2. **風格配對＝原型預填＋同一組滑桿微調**：心理師自評與個案期待用
   **同一組五光譜量尺**，配對可直接算距離；個案先選原型自動預填再微調。
3. **隱私授權與雷點分層**：偏好＝軟性排序加權；雷點＝強降權或不推薦；
   特殊議題標籤（性創傷/家暴/自傷）＝硬性過濾（沿用晤談前 §6）。
4. **試玩小練習**：每位夥伴可標 1 個免費 3–5 分鐘試玩練習，完成後 AI 輕量
   回饋＋預約/收藏 CTA。試玩內容同樣走現有模組審核管線。
5. **AI 媒合理由**：配對顯示相符度＋一句可讀的理由（透明可解釋，不做黑箱分數）。

## 2. 現有使用者預覽的缺口（2026-07-16 盤點）

- 分類單層、服務者視角；心理諮詢類內部無議題劃分，學派藏在模組標題。
- 夥伴呈現僅姓名/職稱/2 tags/一句 bio，無風格、無背景、無媒合資訊量。
- 無搜尋、無狀態導向推薦入口。
- 可沿用的資產：instant/apply 雙模式、credits 錢包、收藏、詳情 sheet。

## 3. 分類改版：白話入口 → 議題標籤對應

| 白話入口（頂層 chips） | 對應議題標籤（晤談前 §6-1） |
|---|---|
| 睡不好 | 睡眠困擾 |
| 情緒好滿 | 情緒困擾（憂鬱/焦慮/恐慌） |
| 壓力爆表 | 壓力調適、職場議題/職業倦怠 |
| 關係卡住 | 人際關係、親密關係、家庭關係 |
| 找不到方向 | 生涯規劃與發展、自我探索 |
| 重大失落 | 創傷議題、悲傷與失落 |
| 想更認識自己 | 自我探索與成長、自我認同 |

- 次篩選列：服務類型四類（小 chips）。
- 「不知道從哪開始」：免費壓力源健檢當預設入口，AI 分析後推薦白話分類。

## 4. 心理師風格檔案（四區塊）

1. **擅長議題**：議題標籤（§6-1），特殊議題硬性過濾。
2. **取向與工作焦點**：取向標籤（§6-2）＋服務族群＋工作焦點
   （情緒整理/行動計畫/關係修復/自我探索）。
3. **多元背景**：跨領域經歷（工程/HR/創業/社工/護理/教育）、語言
   （台語/英語/日語/客語/手語）、性別、多元文化經驗。
4. **性格自評**：五組光譜滑桿＋「我的晤談像…」一句自述。

### 4-1 五組風格光譜（雙方同尺，0–100）

| # | 左端 | 右端 |
|---|---|---|
| 1 | 陪伴傾聽 | 主動指引 |
| 2 | 溫柔婉轉 | 直接明快 |
| 3 | 慢熱深耕 | 快速聚焦 |
| 4 | 感受優先 | 思考優先 |
| 5 | 回家任務很少 | 每週具體任務 |

兩端皆中性描述，無好壞。心理師在個人檔案自評；個案在期待檔案填同一組。

## 5. 個案期待檔案（偏好＋雷點）

- **原型**（預填滑桿用）：溫和陪伴型 / 實務導向型 / 深度探索型 / 還不確定。
- **滑桿微調**：同 §4-1 五組。
- **雷點多選**（觸發強降權/不推薦）：一直被給建議、只是聽沒有具體回饋、
  太多理論術語、一直問過去的事、回家沒有任務會失落、有作業壓力會焦慮。
- 資料來源：媒合流程＝初談表單第 8 節擴充；市集自主瀏覽＝3 題快篩。

## 6. App 呈現

- **夥伴卡片改版**：頭像＋職稱＋擅長議題 chips＋風格光譜迷你圖＋任務量指標
  ＋一句自述＋試玩練習入口。
- **相符度**：「與你的期待相符 87%」＋ AI 一句媒合理由
  （例：「你希望每次都有具體任務，這位夥伴的練習以行動計畫為主」）。
- 雷點違反：不顯示或明確標註原因。
- 行政媒合工作台（/staff）直接重用同一套相符度與理由。

## 7. 試玩小練習機制

- 每位夥伴限 1 個，免費、3–5 分鐘、無需申請，完成後：
  AI 輕量回饋 →「這是◯◯設計的練習，覺得有幫助嗎？」→ 收藏/預約 CTA。
- 轉換漏斗：瀏覽 → 試玩 → 收藏/申請 → 媒合/加入模組。
- 法遵：試玩內容過現有審核管線；AI 回饋沿用禁詞防護。

## 8. 配對計算（規則式，MVP 不用 ML）

- 相符度 = 100 − 五光譜平均距離（正規化）；議題匹配數為前置過濾與加權。
- 雷點 → 對應光譜端點超閾值即強降權。
- AI 只負責把「距離最小且個案最在意的維度」翻譯成一句媒合理由。
