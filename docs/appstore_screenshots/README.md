## 重新擷取
模擬器已登入帳號才有真實內容。流程：
1. `npx cap open ios` → 在模擬器跑起 App（或 `xcrun simctl launch booted com.mindgym.app`）。
2. 走到目標畫面，`xcrun simctl io booted screenshot out.png` 擷取乾淨機身畫面。
3. 用 `Image.resize((1320, 2868))` 放大即可（見當時的一次性指令）。

> 想換尺寸（如 6.7" = 1290 × 2796），改 resize 目標即可。
